
# Can be use in following
- DAGs/Basic Scripts
- NodeJS/Flask/Django
- MYSQL
- Kafka

- To test basic script
```bash
python basic_trace.py
```

- To test on flask app
```bash 
flask run -p 8080
```

- To test on mysql curd operations
```bash 
python mysql-curd.py
```

- To test on kafka
```bash 
python producer.py
```
